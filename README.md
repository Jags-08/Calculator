# 🧮 Calculator App (Python)

A simple command-line **Calculator Application** built using Python. This app performs basic arithmetic operations along with power and square root calculations.

---

## 🚀 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division (with error handling)  
- 🔢 Power calculation  
- √ Square root (with validation)  
- 🔁 Continuous operation using loop  
- ⚠️ Handles invalid inputs and errors  

---

## 🛠️ Tech Stack

- Python 3  
- No external libraries required
